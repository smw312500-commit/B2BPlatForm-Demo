지퍼단추회사 코드 규칙
========================================
작성일: 2026-05-21

========================================
단추 코드 구조
========================================

  소재코드_컬러코드
  예) WOOD_BK = 원목단추 블랙

[ 소재코드 ]
  WOOD    원목 (Wood)
  PLASTIC 플라스틱 (Plastic)
  METAL   금속 (Metal)

[ 컬러코드 ]
  BK    블랙 (Black)
  WH    화이트 (White)
  NV    네이비 (Navy)
  GY    그레이 (Gray)
  BR    브라운 (Brown)   ← 원목 주로 사용
  SV    실버 (Silver)    ← 금속 주로 사용

[ 단추 예시 ]
  WOOD_BR     원목단추 브라운
  WOOD_BK     원목단추 블랙
  PLASTIC_BK  플라스틱단추 블랙
  PLASTIC_WH  플라스틱단추 화이트
  METAL_SV    금속단추 실버
  METAL_BK    금속단추 블랙

========================================
지퍼 코드 구조
========================================

  ZIPPER_사이즈코드
  예) ZIPPER_L = 대형 지퍼

[ 사이즈코드 ]
  S     소형 (Small)   → 티셔츠, 바지 포켓
  M     중형 (Medium)  → 재킷
  L     대형 (Large)   → 다운, 아웃도어

[ 지퍼 예시 ]
  ZIPPER_S    소형 지퍼
  ZIPPER_M    중형 지퍼
  ZIPPER_L    대형 지퍼

========================================
라벨코드 품목코드와 매핑
========================================

  라벨코드 4번째 자리 → 필요 부자재:

  T (티셔츠)  → PLASTIC_** 단추
  P (바지)    → METAL_** 단추 또는 ZIPPER_S
  J (재킷)    → ZIPPER_M + METAL_** 단추
  D (다운)    → ZIPPER_L

========================================
플랫폼 수집 시 item_name 형식
========================================

  DB collected_release 테이블 item_name 컬럼에 입력:
  예) WOOD_BR, ZIPPER_L, PLASTIC_BK
