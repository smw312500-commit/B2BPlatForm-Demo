========================================
케어라벨 코드 규칙
========================================
작성일: 2026-05-21

■ 코드 구조 (총 9자리)

  W  1  W  T  C  01  BK
  │  │  │  │  │   │   └─ 컬러 (2자리)
  │  │  │  │  │   └───── 스타일번호 (01~99)
  │  │  │  │  └───────── 원단코드 (1자리)
  │  │  │  └──────────── 품목코드 (1자리)
  │  │  └─────────────── 성별코드 (1자리)
  │  └────────────────── 계절코드 (1자리)
  └───────────────────── 브랜드코드 (1자리)


========================================
각 자리 상세 규칙
========================================

[ 브랜드코드 - 1자리 ]
  추후 브랜드 추가 시 알파벳으로 확장
  예) W = W브랜드


[ 계절코드 - 1자리 ]
  1 = 봄 (SS 전반기)
  2 = 여름 (SS 후반기)
  3 = 가을 (FW 전반기)
  4 = 겨울 (FW 후반기)


[ 성별코드 - 1자리 ]
  W = 여성 (Women)
  M = 남성 (Men)


[ 품목코드 - 1자리 ]
  T = 티셔츠
  P = 바지 (Pants)
  J = 재킷 (Jacket)
  D = 다운 (Down)


[ 원단코드 - 1자리 ]
  C = 면 (Cotton)
  P = 폴리에스터 (Polyester)
  L = 린넨 (Linen)
  W = 울 (Wool)
  M = 혼방 (Mixed)


[ 스타일번호 - 2자리 ]
  01~99
  동일 브랜드/계절/성별/품목/원단 조합 내에서 디자인 구분


[ 컬러코드 - 2자리 ]
  BK = 블랙 (Black)
  WH = 화이트 (White)
  NV = 네이비 (Navy)
  GY = 그레이 (Gray)
  BE = 베이지 (Beige)
  RD = 레드 (Red)
  추후 확장 가능


========================================
예시
========================================

  W1WTC01BK
  → W브랜드 / 봄 / 여성 / 티셔츠 / 면 / 01번 스타일 / 블랙

  W3MJW01NV
  → W브랜드 / 가을 / 남성 / 재킷 / 울 / 01번 스타일 / 네이비

  W4MDP01BK
  → W브랜드 / 겨울 / 남성 / 다운 / 폴리에스터 / 01번 스타일 / 블랙


========================================
플랫폼 연동 포인트
========================================

  이 라벨코드는 부자재 3사를 연결하는 핵심 키다.

  원단코드(5번째 자리) → 옷감회사 원단 종류와 매핑
  품목코드(4번째 자리) → 지퍼/단추회사 품목과 매핑
    - 재킷(J), 다운(D) → 지퍼 필요
    - 티셔츠(T), 바지(P) → 단추 필요 (품목에 따라 다름)

  AI 에이전트는 라벨코드 하나로
  옷감사 + 지퍼단추사 + 물류사 출고 현황을 연결하여
  납기 위험 감지 및 자동화 트리거에 활용한다.
