========================================
물류회사 Agent UI 구조 명세
========================================
작성일: 2026-05-21

이 파일을 읽는 AI에게:
스케치 2개(jpg)를 먼저 확인하고 이 문서를 읽을 것.
물류 Agent UI는 생산사 UI와 구조가 다름.
생산사: AI Agent 패널이 우측 고정
물류사: AI 지시 영역이 하단에 위치 (스케치2 참고)

========================================
전체 레이아웃
========================================

```
┌─────────────────────────────────────────────────────┐
│  물류 Agent                          날짜/시간       │
├─────────────────────────────────────────────────────┤
│  [ 기사관리 ]  [ 화물관리/배차관리 ]  [ 기타 ]      │
├─────────────────────────────────────────────────────┤
│                                                     │
│                                                     │
│              메인 콘텐츠 영역                        │
│           (탭 클릭 시 내용 전환)                    │
│                                                     │
│                                                     │
├─────────────────────────────────────────────────────┤
│                  AI 지시                            │
└─────────────────────────────────────────────────────┘
```

- 메인 영역: 전체 높이의 약 75%
- AI 지시 영역: 하단 고정 (탭 바뀌어도 항상 표시)
- ※ 생산사와 달리 AI 영역이 우측이 아닌 하단에 위치

========================================
탭 구성 (3개)
========================================

| 순서 | 탭명 | 설명 |
|------|------|------|
| 1 | 기사관리 | 기사 등록/현황, 차량 등록/현황 |
| 2 | 화물관리/배차관리 | 화물 목록, 배차 현황, 왕복 최적화 |
| 3 | 기타 | 설정 등 |

========================================
탭별 화면 상세
========================================

### [ 기사관리 ] 탭

좌측: 기사 목록
```
기사명 | 연락처 | 현재위치(시/구) | 차량번호 | 상태(가용/운행중/휴무)
[ 기사 등록 버튼 ]
```

우측: 차량 목록
```
차량번호 | 기사명 | 최대적재량(kg) | 차량종류 | 상태
[ 차량 등록 버튼 ]
```

---

### [ 화물관리/배차관리 ] 탭 (핵심)

화물 목록 테이블 (스케치1 기준):
```
집하주소   | 가는길  | 빈차일지  | 납기일 | 배송현황  | 왕복최적화
(출발회사) | (경로)  | (빈차여부)| (날짜) | (상태)    | (AI 매칭)
```

컬럼 상세:
  집하주소    출발 회사 주소 (platform DB 회사정보에서 자동 조회)
  가는길      출발지 → 도착지 (인천항/부산항)
  빈차일지    귀환 시 빈차 여부 / 연결 화물 있으면 표시
  납기일      화물 납기일
  배송현황    배차대기 / 운행중 / 완료
  왕복최적화  AI가 귀환 경로에 연결 가능한 화물 표시

---

### [ 기타 ] 탭
```
기본 설정 (이동시간 기준 등)
```

========================================
하단 AI 지시 영역
========================================

```
┌─────────────────────────────────────────────────────┐
│  AI 지시                                            │
│                                                     │
│  [오늘 배차] 김기사 → 부산 사하구 픽업 → 인천항     │
│  [왕복연결] 인천항 → 부산 해운대구 픽업 가능         │
│  [납기위험] W3MJW01NV 납기 D-1 긴급 배차 필요       │
│                                                     │
└─────────────────────────────────────────────────────┘
```

표시 내용:
  - 오늘 배차 스케줄 요약
  - 왕복 최적화 연결 가능 화물
  - 납기 위험 긴급 알림
  - 빈차 운행 경고
