========================================
지퍼단추agent 실행 가이드
========================================

========================================
1. 사전 준비
========================================

  - Python 3.11+
  - Node.js 18+
  - MySQL 서버 실행 중
  - company_zipper 스키마는 백엔드 최초 실행 시 자동 생성됩니다.

========================================
2. 간편 실행 (권장)
========================================

  start.bat 더블클릭
  → 백엔드(8003) + 프론트엔드(5175) 자동 실행
  → 브라우저 자동 오픈: http://localhost:5175

========================================
3. 수동 실행
========================================

  [ 백엔드 ]
  cd 지퍼단추agent\backend
  copy .env.example .env
  # .env 파일에서 DB 접속정보 / OPENAI_API_KEY 입력
  pip install -r requirements.txt
  uvicorn main:app --reload --port 8003

  API 문서: http://localhost:8003/docs

  [ 프론트엔드 ]
  cd 지퍼단추agent\frontend
  npm install
  npx vite

  브라우저: http://localhost:5175

========================================
4. DB 테이블 자동 생성
========================================

  백엔드 최초 실행 시 SQLAlchemy가 자동으로
  아래 테이블을 생성합니다:
    - zipper_stock   (원자재 재고)
    - zipper_order   (발주)
    - zipper_release (출고/생산)

  스키마 초기화가 필요할 때:
    cd backend
    python init_db.py

========================================
5. 포트 정리
========================================

  Backend:  8003
  Frontend: 5175
  MySQL:    3306 (기본값)

  라벨agent Backend:  8001
  옷감agent Backend:  8002

========================================
6. OPENAI_API_KEY 없이 실행
========================================

  .env에 OPENAI_API_KEY가 없어도 동작합니다.
  AI 분석 기능은 규칙 기반으로만 실행되며
  GPT 코멘트는 표시되지 않습니다.
