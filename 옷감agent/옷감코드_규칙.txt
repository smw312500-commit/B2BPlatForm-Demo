========================================
옷감회사 코드 규칙
========================================
작성일: 2026-05-21

■ 라벨코드와의 연동 포인트
  라벨코드 5번째 자리(원단코드)와 옷감코드가 매핑된다.
  예) 라벨코드 W1WTC01BK → 원단코드 C → 면 원단 출고 필요


========================================
원단 종류 (5종)
========================================

  코드  원단명          특징
  ────────────────────────────────────────
  C     면 (Cotton)     기본 소재, 사계절 활용
  P     폴리에스터      경량·기능성, 아웃도어/다운류
  L     린넨 (Linen)    여름 소재, 통기성 우수
  W     울 (Wool)       겨울·고급 라인, 재킷/코트류
  M     혼방 (Mixed)    소재 2종 이상 혼합


========================================
컬러 코드
========================================

  코드  컬러
  ────────────────────
  BK    블랙 (Black)
  WH    화이트 (White)
  NV    네이비 (Navy)
  GY    그레이 (Gray)
  BE    베이지 (Beige)
  RD    레드 (Red)
  추후 확장 가능


========================================
입출고 데이터 항목
========================================

  - 원단코드 (C/P/L/W/M)
  - 컬러코드
  - 입고량 (단위: 야드)
  - 출고량 (단위: 야드)
  - 현재 재고량
  - 날짜
  - 연동 라벨코드 (출고 시 어떤 주문 건인지 기록)


========================================
품목-원단 연관 관계
========================================

  품목(라벨코드 4번째 자리)과 원단의 자연스러운 매핑:

  T (티셔츠)  → 주로 C(면), P(폴리에스터)
  P (바지)    → 주로 C(면), M(혼방)
  J (재킷)    → 주로 W(울), M(혼방)
  D (다운)    → 주로 P(폴리에스터)

  ※ 강제 제약은 아님. AI 에이전트가 패턴으로 학습.


========================================
AI 인사이트 활용 예시
========================================

  ① W(울) 출고 급증 + J(재킷) 라벨 증가
     → "고급 울 재킷 수요 증가 → 울 재고 선제 확보 권고"

  ② P(폴리에스터) 대량 출고 + D(다운) 라벨 급증
     → "아웃도어 다운 시즌 수요 → 관련 부자재 준비 권고"

  ③ 특정 원단 재고 부족 + 납기 임박 라벨 존재
     → "납기 지연 위험 감지 → 긴급 발주 알림"
