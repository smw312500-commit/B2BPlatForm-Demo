========================================
4. AI Agent 계산 로직
========================================

  [ 소요시간 계산 ]
  소요시간(h) = 주문량(장) ÷ (가동 인쇄기 대수 × 800)

  예) 라벨 7,200장 주문, 인쇄기 6대 가동 시:
      7,200 ÷ (6 × 800) = 1.5시간 → 당일 생산 가능

  [ 필요 원자재 계산 ]
  필요 라벨원단(m) = 주문량(장) ÷ 25
  필요 잉크(통)   = 주문량(장) ÷ 10,000 (소수점 올림)

  예) 라벨 7,200장 주문 시:
      원단 = 7,200 ÷ 25 = 288m 필요
      잉크 = 7,200 ÷ 10,000 = 1통 필요

  [ 납기 가능 여부 ]
  납기 가능일수 = 소요시간 ÷ 9시간(일일 가동)
  납기일 > 오늘 + 납기 가능일수 → ✅ 납기 가능
  납기일 ≤ 오늘 + 납기 가능일수 → ⚠ 납기 위험
